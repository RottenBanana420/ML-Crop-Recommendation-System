# Documentation Directory

This directory contains all project documentation organized by category.

## Structure

### `/planning/`
Contains project planning documents, implementation roadmaps, and update summaries:
- `FLASK_IMPLEMENTATION_ROADMAP.md` - Roadmap for Flask web application implementation
- `GITIGNORE_UPDATE_SUMMARY.md` - Summary of .gitignore updates
- `README_UPDATE_SUMMARY.md` - Summary of README updates
- `UI_DESIGN.md` - Detailed UI design specifications
- `UI_DESIGN_INDEX.md` - Index of UI design components
- `UI_DESIGN_SUMMARY.md` - Summary of UI design decisions

### `/design/`
Contains UI/UX design files, mockups, and design references:
- `mockups/` - Visual mockups and design assets
- `MOCKUPS_REFERENCE.md` - Reference documentation for mockups
- `README.md` - Design directory overview
